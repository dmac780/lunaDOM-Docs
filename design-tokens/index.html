Design Tokens

Built-in grid system and design tokens

lunaDOM includes tokens.css — a lightweight grid system and spacing scale for rapid prototyping.

Installation

Include the tokens stylesheet in your HTML:

<link rel="stylesheet" href="node_modules/lunadom/tokens.css">

Grid System

Use HTML attributes to build responsive layouts. No CSS classes needed.

<!-- Basic grid --> <div grid cols="3" gap="4"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> </div> <!-- Auto-fit grid (responsive) --> <div grid min="20" gap="6"> <div>Auto Card</div> <div>Auto Card</div> <div>Auto Card</div> </div> <!-- Responsive breakpoints --> <div grid cols="1" cols-md="2" cols-lg="4" gap="4"> <div>Mobile: 1 col</div> <div>Tablet: 2 cols</div> <div>Desktop: 4 cols</div> </div>

Available Attributes

Grid Container

grid Enable grid layout
cols="3" Fixed columns (1-6, 12)
cols-md="2" Tablet (768px+)
cols-lg="4" Desktop (1024px+)
min="20" Auto-fit min width (rem)
gap="4" Spacing (0-6 scale)

Grid Items

span="2" Span columns (1-12)
span="full" Full width
row="2" Span rows (1-3)

Spacing Scale

--space-1 0.25rem
--space-3 1rem
--space-6 3rem

Code Example

<!DOCTYPE html> <html> <head> <!-- Include tokens.css --> <link rel="stylesheet" href="node_modules/lunadom/tokens.css"> </head> <body> <!-- Dashboard layout --> <div grid cols="1" cols-lg="3" gap="6"> <!-- Main area (2 cols on desktop) --> <div span="2"> <h2>Main Content</h2> <!-- Nested auto grid --> <div grid min="16" gap="4"> <div>Card 1</div> <div>Card 2</div> <div>Card 3</div> </div> </div> <!-- Sidebar (1 col) --> <div>Sidebar</div> </div> </body> </html>
