Tree

Hierarchical tree view with selection and lazy loading.

The <luna-tree> and <luna-tree-item> components display hierarchical tree structures with selection modes and lazy loading.

Paths

/lunadom/components/tree/tree.js REQUIRED
/lunadom/components/tree-item/tree-item.js REQUIRED

Overrides

@slot default - Tree item content
@slot children - Child tree items
@slot expand-icon - Custom expand icon
@slot collapse-icon - Custom collapse icon
@attr selection - Selection mode (leaf, multiple, single)
@attr expanded - Initially expanded state
@attr selected - Initially selected state
@attr disabled - Disable the item
@attr lazy - Enable lazy loading
--indent-guide-width - Indent guide line width
--indent-guide-color - Indent guide line color
