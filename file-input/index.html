File Input

Drag-and-drop file upload with multiple file support.

The <luna-file-input> component provides drag-and-drop file upload with support for multiple files and custom file type restrictions.

Paths

/lunadom/components/file-input/file-input.js REQUIRED

Overrides

@slot dropzone - Custom content for the drop zone
@attr label - Label text for the input
@attr hint - Help text below the input
@attr multiple - Allow multiple file selection
@attr accept - Accepted file types (e.g., ".pdf,.doc")
luna-file-change - Emitted when files are selected
