Card

Flexible containers for content with headers, images, and footers.

The <luna-card> component provides flexible containers for displaying content with optional headers, images, and footers.

Paths

/lunadom/components/card/card.js REQUIRED

Overrides

@slot image - Image for the card
@slot header - Header content for the card
@slot default - Main content for the card
@slot footer - Footer content for the card
@attr glass - Whether the card has glass effect
--luna-card-bg - Card background color
--luna-card-color - Card text color
--luna-card-border - Card border color
