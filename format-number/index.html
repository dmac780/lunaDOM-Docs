Format Number

Format numeric values with locale support.

The <luna-format-number> component formats numeric values using the browser's native Intl.NumberFormat API with automatic locale detection.

Paths

/lunadom/components/format-number/format-number.js REQUIRED

Overrides

@attr value - Number to format (required)
@attr type - Format type (decimal, currency, percent, unit, default: "decimal")
@attr currency - ISO 4217 currency code (e.g., "USD", "EUR", "GBP")
@attr currency-display - Currency label style (symbol, narrowSymbol, code, name, default: "symbol")
@attr unit - Physical unit identifier (e.g., "kilometer-per-hour", "celsius")
@attr unit-display - Unit label style (long, short, narrow, default: "short")
@attr min-fraction-digits - Minimum decimal places
@attr max-fraction-digits - Maximum decimal places
@attr no-grouping - Disable thousands separator
@attr notation - Number notation (standard, scientific, engineering, compact, default: "standard")
@attr compact-display - Compact suffix length (short, long, default: "short")
@attr sign-display - Sign visibility (always, exceptZero, never, auto, default: "auto")
