Button

Interactive buttons with multiple variants, sizes, and states.

The <luna-button> component provides interactive buttons with customizable styles and behaviors.

Paths

/lunadom/components/button/button.js REQUIRED

Overrides

@slot default - Content for the button
@attr variant="default" - Variant of the button (default, primary, success, warning, danger, neutral)
@attr size="md" - Size of the button (sm, md, lg)
@attr pill - Whether the button has pill shape
@attr outline - Whether the button has outline style
@attr loading - Whether the button shows loading state
@attr disabled - Whether the button is disabled
@attr caret - Whether the button shows a caret icon
--luna-button-bg - Button background color
--luna-button-color - Button text color
