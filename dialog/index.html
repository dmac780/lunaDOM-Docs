Dialog

Modal dialog component with overlay and focus trapping.

The <luna-dialog> component displays modal dialogs with overlay, focus trapping, and smooth animations.

Paths

/lunadom/components/dialog/dialog.js REQUIRED

Overrides

@slot title - Title content for the dialog
@slot default - Main content for the dialog
@slot footer - Footer content for actions
@method show() - Show the dialog
@method hide() - Hide the dialog
--luna-dialog-overlay - Overlay background color
--luna-bg - Dialog background color
--luna-color - Dialog text color
--luna-border - Dialog border color

Prevent Flash of Dialogs

<style> luna-dialog:not(:defined) { display: none !important; } </style>
