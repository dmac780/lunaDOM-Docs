Range

Slider input for selecting numeric values.

The <luna-range> component provides a slider input for selecting numeric values with tooltip and custom styling.

Paths

/lunadom/components/range/range.js REQUIRED

Overrides

@attr label - Label text
@attr min - Minimum value
@attr max - Maximum value
@attr value - Current value
@attr step - Step increment
@attr help-text - Help text below slider
@attr tooltip-placement - Tooltip position (top, bottom)
@attr disabled - Disable the range
--luna-range-track-active-color - Active track color
--luna-range-track-color - Track background color
--luna-range-thumb-border - Thumb border color
--luna-range-track-height - Track height

Example Code

<!-- Standard Range --> <luna-range label="Volume" min="0" max="100" value="50" help-text="Adjust the system volume"> </luna-range> <!-- Step & Tooltip Placement --> <luna-range label="Rating" min="0" max="10" step="0.5" value="7.5" tooltip-placement="bottom"> </luna-range> <!-- Custom Styles --> <luna-range label="Probability" value="30" style="--luna-range-track-active-color: #f59e0b; --luna-range-track-height: 4px;"> </luna-range> <!-- Disabled State --> <luna-range label="Locked Setting" value="40" disabled></luna-range>
