Select

Dropdown select with single and multiple selection modes.

The <luna-select> component provides dropdown select with single and multiple selection, icons, and grouping support.

Paths

/lunadom/components/select/select.js REQUIRED
/lunadom/components/option/option.js REQUIRED
/lunadom/components/option-group/option-group.js REQUIRED

Overrides

@slot default - <luna-option> or <luna-option-group> elements
@slot prefix - Icon before the select
@attr label - Label text
@attr placeholder - Placeholder text
@attr value - Selected value(s)
@attr multiple - Allow multiple selections
@attr clearable - Show clear button
@attr size - Size (sm, md, lg)
@attr disabled - Disable the select
@attr help-text - Help text below select
@attr max-options-visible - Max options to display in dropdown
