Details

Collapsible accordion component with grouping support.

The <luna-details> component provides collapsible sections with accordion grouping support.

Paths

/lunadom/components/details/details.js REQUIRED

Overrides

@slot summary - Summary/header text
@slot icon - Icon before the summary
@slot default - Content to show when expanded
@attr open - Whether initially open
@attr group - Group name for accordion behavior
@attr disabled - Whether interaction is disabled
luna-toggle - Emitted when expanded/collapsed
