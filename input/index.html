Input

Text input with variants, decorators, and validation.

The <luna-input> component provides text inputs with multiple variants, decorators, and built-in validation support.

Paths

/lunadom/components/input/input.js REQUIRED

Overrides

@slot prefix - Content before the input
@slot suffix - Content after the input
@attr label - Label text
@attr placeholder - Placeholder text
@attr value - Input value
@attr type - Input type (text, password, email, number, date, etc.)
@attr variant - Style variant (default, filled)
@attr size - Size (sm, md, lg)
@attr clearable - Show clear button
@attr toggle - Show password toggle (for password type)
@attr disabled - Disable the input
@attr readonly - Make the input read-only
@attr help-text - Help text below the input
--luna-input-bg - Input background color
--luna-input-border - Input border color
