Combobox

Autocomplete select with filtering, custom values, and grouping.

The <luna-combobox> component provides an autocomplete select with filtering, custom values, multiple selection, and grouped options.

Paths

/lunadom/components/combobox/combobox.js REQUIRED
/lunadom/components/option/option.js REQUIRED
/lunadom/components/option-group/option-group.js REQUIRED

Overrides

@slot default - <luna-option> or <luna-option-group> elements
@attr label - Label text for the combobox
@attr placeholder - Placeholder text
@attr autocomplete - Enable autocomplete filtering
@attr allow-custom-value - Allow entering custom values
@attr multiple - Allow multiple selections
@attr clearable - Show clear button
@attr help-text - Help text below the combobox
@attr max-options-visible - Max options to show in dropdown
