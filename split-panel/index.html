Split Panel

Resizable split panels with horizontal and vertical layouts.

The <luna-split-panel> component creates resizable split panels with snapping, nesting, and custom divider styling.

Paths

/lunadom/components/split-panel/split-panel.js REQUIRED

Overrides

@slot start - Content for start/left/top panel
@slot end - Content for end/right/bottom panel
@attr position - Initial split position (px or %)
@attr vertical - Use vertical split instead of horizontal
@attr primary - Which panel is fixed (start, end)
@attr snap - Snap positions (e.g., "repeat(5, 20%)")
--divider-width - Divider thickness
--divider-color - Divider background color
--handle-color - Drag handle color
--handle-size - Drag handle size
